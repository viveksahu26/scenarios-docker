## Write hello world program

#### Crete new directory

    mkdir hello
#### Create file with name main.go

    vim main.go
#### paste the below code,

```
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```
#### Execute the program

    go run main.go
